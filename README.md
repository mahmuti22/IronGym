# IronGym

Premium dark landing for **IronGym** — Next.js (App Router), Tailwind CSS v4, Framer Motion.

## Scripts

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 15
- React 19
- Tailwind CSS 4 (`@tailwindcss/postcss`)
- Framer Motion (section reveals + hero)
